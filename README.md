## 📖 Історія проекту

Цей проект народився не просто як програмне забезпечення, а як рішення реальної проблеми в українській школі під час війни.

Мій батько працює вчителем фізики. Він завжди прагнув до автоматизації та інновацій, тому спочатку намагався налаштувати систему шкільних дзвінків через розумні реле Sonoff. Це працювало, але реальність внесла свої корективи. Повітряні тривоги, постійні перегляди розкладів через відключення світла та необхідність оперативно змінювати час уроків зробили використання звичайних IoT-рішень занадто складним. Систему доводилося постійно вимикати, і вчителям знову доводилося давати дзвінки вручну.

Тоді ми вирішили створити **School Bell** — спеціалізовану програму, яка дозволяє гнучко керувати розкладом уроків та автоматично подавати дзвінки через реле, підключене до комп'ютера. Навіть у найскладніші часи технології мають допомагати освіті залишатися організованою.

## 🚀 Основні можливості

- **Гнучке керування розкладами:** Створення кількох варіантів розкладів (стандартний, скорочений, святковий).
- **База даних:** Збереження інформації про вчителів, предмети та класи за допомогою SQLite.
- **Апаратна інтеграція:** Керування фізичним шкільним дзвінком через USB-реле.
- **Зручний інтерфейс:** Інтуїтивно зрозуміле вікно для швидкого редагування часу уроків та перерв.
- **Перспективи:** Планується додавання трансляції розкладу на телевізори (через WebSockets) та сучасний дизайн карток.

## 💻 Системні вимоги

Для стабільної роботи програми рекомендуються наступні параметри:

- **Операційна система:** Windows 10 або Windows 11 (64-bit).
- **Процесор:** Intel Core i3 / AMD Ryzen 3 або аналогічний (2 ядра, 2.0 ГГц+).
- **Оперативна пам'ять:** від 4 ГБ.
- **Вільне місце на диску:** ~200 МБ.
- **Порти:** 1 вільний USB-порт для підключення USB-реле (керування дзвінком).
- **Дисплей:** роздільна здатність від 1280x720 для комфортної роботи з графіком.

## 📦 Встановлення та запуск

Програма поширюється як самодостатній додаток для Windows:

1. **Завантаження:** Отримайте архів з програмою.
2. **Розпакування:** Розпакуйте папку `SchoolBell` у зручне для вас місце (наприклад, на Робочий стіл або диск C:).
3. **Запуск:** Відкрийте папку та запустіть файл `SchoolBell.exe`.
4. **Важливо:** Не видаляйте папки `app` та `runtime`, що знаходяться поруч із виконуваним файлом — вони необхідні для запуску системи без встановленої Java на комп'ютері.
