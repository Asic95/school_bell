# 🗺 Огляд системи School Bell

Цей документ містить опис поточної структури проекту, функціональних можливостей та взаємозв'язків між модулями. Він служить довідником для розробки та підтримки системи.

## 🏗 Архітектура
- **Мова:** Java 21
- **UI Фреймворк:** JavaFX (з використанням Soft UI/Material 3 дизайну)
- **База даних:** SQLite (`school_bell.db`)
- **Збірка:** Maven
- **Веб-компонент:** Вбудований сервер для трансляції розкладу (підтримка тем `classic`, `modern` та нової висококонтрастної `cyber` через WebSockets).

## 📂 Структура папок та ключові класи

### 🧠 Сервіси (`com.schoolbell.service`)
Це "мозок" програми, де зосереджена вся бізнес-логіка:
- **`DatabaseManager`**: Керує підключенням до SQLite та міграціями (підтримує оновлення схеми для кабінетів).
- **`ScheduleService`**: Логіка розкладу дзвінків, визначення поточного уроку/перерви.
- **`SignalService`**: Керування подачею сигналів (дзвінків) через реле. Включає режим "Хвилина мовчання" о 09:00, систему затримки стану до завершення сигналу та інструменти діагностики.
- **`AcademicService`**: Управління даними про вчителів, предмети, класи, аудиторії та заміни.
- **`AudioService`**: Відтворення аудіо-файлів (мелодій дзвінків) з підтримкою пресетів гучності.
- **`BroadcastService`**: Керування трансляцією даних на зовнішні екрани/веб-панелі.
- **`ConfigService`**: Збереження налаштувань програми.

### 🖥 Інтерфейс Користувача (`com.schoolbell.ui`)
- **`MainApp`**: Точка входу. Керує глобальним **Журналом системи** (ObservableList), який збирає всі події в реальному часі.
- **`DashboardView`**: Головний екран. Оновлено блок керування звуком на **Smart Presets** (0-100%). Кнопки екстрених сигналів тепер блокуються під час програвання сигналу для запобігання конфліктам.
- **`EfirView`**: Центр керування ефіром. Включає новий візуальний **Журнал системи** з кольоровим маркуванням статусів (Успіх, Попередження, Помилка).
- **`SystemView`**: Глобальні налаштування. Перероблений у формат **"Setting Tiles"** з великими іконками та покращеним контрастом тексту. Включає новий **Signal Pattern Editor** з горизонтальною живою візуалізацією ритму дзвінків.
- **`NotificationsView`**: Центр керування сповіщеннями. Використовує систему пресетів гучності для стабільної взаємодії з Windows.
- **`SignalsView`**: Налаштування фізичних сигналів реле. Повністю синхронізований з редактором у налаштуваннях системи.
- **`UIStyles` / `UIComponents`**: Набір сучасних SVG-іконок (`ICON_POWER`, `ICON_FLASK`, `ICON_WAVEFORM`) та покращені картки керування з підтримкою багаторядкового тексту та адаптивною висотою.
- **`ScheduleEditorDialog`**: Основне вікно редагування з уніфікованими макетами (горизонтальні розділювачі, ScrollPane з FitToWidth) та синхронізованими колірними темами для кожного розділу:
    - `ClassroomsEditorTab`: Бірюзова тема (`#00cec9`).
    - `SubstitutionsEditorTab`: Помаранчева тема (`#e67e22`).
    - `BellsEditorTab`: Синя тема (`#0984e3`).

### 🔌 Апаратна частина (`com.schoolbell.hardware`)
- **`RelayController`**: Взаємодія з фізичним USB-реле.

## 🛠 Поточний функціонал
1.  **Автоматичні дзвінки:** Програма вираховує час до наступної події та активує реле.
2.  **Живий моніторинг:** Візуальний журнал усіх подій системи (включаючи симуляцію) у розділі «Ефір».
3.  **Гнучкий розклад:** Швидка зміна часу уроків з миттєвою візуалізацією «малюнку» дзвінка.
4.  **Система замін:** Відображення актуальної інформації про заміни та формування щомісячних звітів.
5.  **Розумне керування звуком:** Відмова від нестабільних слайдерів на користь фіксованих ступенів гучності (пресетів).
6.  **Центр керування ефіром:** Повний контроль над трансляцією з візуальною індикацією статусу та списком підключених моніторів.
7.  **Адаптивність та Доступність:** Збільшені шрифти, покращений контраст та динамічні картки для зручної роботи на будь-яких екранах.
