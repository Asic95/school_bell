# 🗺 Огляд системи School Bell (v1.0)

Цей документ містить опис архітектури та технічної структури проекту School Bell після переходу до фінальної версії 1.0.

## 🏗 Архітектура
- **Мова:** Java 21
- **UI Фреймворк:** JavaFX (Дизайн-система **Modern Academic Soft UI v2**)
- **База даних:** SQLite. Файл бази `school_bell.db` перенесено до `%APPDATA%\SchoolBell\` для безпечного зберігання та збереження даних при оновленнях.
- **Веб-компонент:** Вбудований HTTP/WebSocket сервер для трансляції розкладу (дефолтний порт: **9090**).

## 📂 Ключові компоненти та сервіси

### 🖥 Інтерфейс та керування вікном (`com.schoolbell.ui`)
- **`TitleBar`**: Кастомна панель заголовка. Замінює стандартну рамку ОС, забезпечує перетягування вікна, максимізацію (з урахуванням Visual Bounds) та стилізовані кнопки керування.
- **`TrayManager`**: Керує іконкою в системному треї, контекстним меню та згортанням програми біля годинника.
- **`UIStyles`**: Централізована система стилів. Використовує змінні для кольорів, тіней та радіусів (28px для карток, 18px для кнопок).
- **`DashboardView`**: Головний екран із розумною логікою відображення поточного стану дня.

### 🧠 Бізнес-логіка та система (`com.schoolbell.service`)
- **`PathService`**: Визначає шляхи до папок користувача в системі. Забезпечує коректну роботу в папках із обмеженими правами (наприклад, Program Files).
- **`DatabaseManager`**: Керує з'єднанням з базою даних, ініціалізацією та автоматичною міграцією даних зі старих версій програми.
- **`AudioService`**: Забезпечує плавне відтворення аудіо з використанням ефектів **Fade-in** та **Fade-out** (3 секунди) для м'яких переходів. Підтримує MP3 та прямий вивід WAV через мікшер.
- **`SignalService`**: Виконує розклад дзвінків, керує реле, підтримує режими симуляції та екстрених сповіщень.
- **`ScheduleService`**: Генерує стандартизовані події з префіксами **"IN:"** та **"OUT:"**, що гарантує надійне розпізнавання перерв для автоматизації медіа-ефіру.
- **`MediaSchedulerService`**: Керує фоновою музикою та оголошеннями, прив'язаними до розкладу або конкретних перерв.
- **`AirAlertService`**: Автоматичний моніторинг повітряних тривог через API та інтеграція з системою дзвінків.
- **`SystemService`**: Виконує низькорівневі налаштування Windows: автозапуск (через реєстр), правила Firewall для порту 9090, керування системною гучністю.

### 🔐 Захист та запуск (`com.schoolbell`)
- **`InstanceGuard`**: Запобігає запуску кількох копій програми. Якщо програма вже запущена, нова копія просто активує вже існуюче вікно.
- **`Launcher`**: Точка входу, що забезпечує коректне завантаження JavaFX середовища.

## 📦 Розповсюдження (Distribution)
- **`build_dist.ps1`**: Автоматизований PowerShell скрипт для повної збірки проекту.
    - Компіляція Maven.
    - Генерація мульти-форматної іконки (`.ico`).
    - Створення App Image через `jpackage`.
    - Компіляція інсталятора через Inno Setup.
