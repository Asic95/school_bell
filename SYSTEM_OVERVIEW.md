# 🗺 Огляд системи School Bell

Цей документ містить опис поточної структури проекту, функціональних можливостей та взаємозв'язків між модулями. Він служить довідником для розробки та підтримки системи.

## 🏗 Архітектура
- **Мова:** Java 21
- **UI Фреймворк:** JavaFX (Дизайн-система **Modern Academic Soft UI v2**)
- **База даних:** SQLite (`school_bell.db`)
- **Збірка:** Maven
- **Веб-компонент:** Вбудований сервер для трансляції розкладу (теми `classic`, `modern`, `cyber`, `soft`).

## 📂 Структура папок та ключові класи

### 🖥 Інтерфейс Користувача (`com.schoolbell.ui`)
Система використовує централізовану фабрику компонентів для забезпечення візуальної цілісності.

- **`UIStyles` / `ControlFactory`**: Фундамент дизайну. Повна відмова від магічних значень кольорів на користь категорій `GLASS_`, `SHADOW_`, `BORDER_` та `TR_`.
    - `SOFT_CARD`: Базова поверхня (28px радіус, м'які Navy-тіні).
    - `PREMIUM_FIELD_ERROR_STYLE`: Візуальна валідація порожніх полів.
    - `createToggleSwitch`: Розумні перемикачі з контекстною анімацією.
- **`AppNavigation`**: Керування переходами. Використовує `FadeTransition` для плавного перемикання між розділами без моргання контенту.
- **`DashboardView`**: Головний екран. 
    - **Smart Day Logic**: Коректне відображення стану "Очікування" до початку першого уроку.
    - **Contextual Relay Status**: Явне відображення "Режиму симуляції" з кольоровим кодуванням.
- **`EfirView`**: Центр трансляції. Керування WebSocket-сервером та налаштуванням веб-дашборду.
- **`MediaEventEditorDialog`**: Покращений віджет вибору медіа-джерел (File/Folder Badge) з розумним скороченням довгих шляхів.

### 🧠 Сервіси (`com.schoolbell.service`)
- **`SignalService`**: Керування реле. Підтримка симуляції, "Хвилини мовчання" та екстрених сигналів.
- **`AirAlertService`**: Інтелектуальний моніторинг повітряних тривог. Забезпечує автоматичне сповіщення та інтеграцію з візуальною системою попередження.
- **`AnnouncementService`**: Керування базою активних та архівованих оголошень.
- **`ConfigService`**: Глобальне сховище налаштувань.
- **`SystemService`**: Низькорівневі операції Windows (реєстр, автозапуск EXE, Firewall, системна гучність).
- **`DatabaseManager`**: Керує SQLite, підтримує міграції схеми даних.

### 🔌 Апаратна частина (`com.schoolbell.hardware`)
- **`RelayController`**: Розумний драйвер USB-реле. Автоматично перемикається в режим логування при увімкненні симуляції.

## 🛠 Поточний функціонал
1.  **Преміальний Soft UI v2:** Повна уніфікація інтерфейсу. Використання централізованих констант кольорів, радіусів 28px/18px та плавних анімацій переходів.
2.  **Інтелектуальний розклад:** Автоматичне визначення поточного контексту та підтримка декількох варіантів розкладів.
3.  **Екстрена безпека:** Автоматизована реакція на повітряні тривоги через `AirAlertService`, кастомізація звуків для різних типів небезпек.
4.  **Повний контроль ефіру:** Моніторинг підключених табло, керування темами оформлення (включаючи нову світлу тему `soft`).
5.  **Адаптивність вводу:** Візуальна валідація полів та спеціалізовані контроли для часу та медіа-джерел.
